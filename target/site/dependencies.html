jdbc-can-be-nice

This 'toy' library is not intended for large scale projects, but rather for quick hacking with jdbc, making it less painful and more pleasant to use by exposing a fluent API that hides the awful SQLException handling code and allows for easy chaining of operations

http://github.com/jawher/jdbc-can-be-nice
